================================================================================
INSTRUCCIONES PARA GENERAR PRESENTACIÓN (PPT) CON python-pptx
================================================================================

OBJETIVO:
Genera una presentación profesional en PowerPoint (.pptx) usando la librería
python-pptx. La presentación es para el cliente Círculo de Crédito y debe
comunicar la propuesta técnica de un agente de atención 24/7 para Otorgantes
de Crédito.

DISEÑO Y ESTILO VISUAL
================================================================================

REFERENCIA DE DISEÑO:
- Estilo inspirado en Anthropic (anthropic.com): minimalista, elegante, degradados
  en tonos naranja/ámbar/crema.

IMÁGENES DE REFERENCIA (revisar todas para capturar el estilo completo):
  1. Artifacts/WorkSheet.png
     → Cronograma del proyecto. Usa paleta naranja sobre fondo claro.
       Referencia para barras, headers y tipografía.
  2. Artifacts/referenceStyle1.png
     → Screenshot de anthropic.com o diseño de referencia Anthropic.
       Referencia para degradados, layout general y estética minimalista.
  3. Artifacts/referenceStyle2.png
     → Screenshot adicional de anthropic.com o material Anthropic.
       Referencia para paleta de colores, cards, y estilo de presentación
       de información.

  INSTRUCCIÓN: Analiza las 3 imágenes de referencia y replica el estilo visual
  (degradados, espaciado, proporciones, contraste) en todas las diapositivas.
  No copies el contenido, solo el estilo.

PALETA DE COLORES:
- Naranja principal: #FF6B35 (accent, títulos de sección, barras)
- Naranja oscuro: #E85D26 (degradados, hover)
- Naranja suave/crema: #FFF3EC (fondos, backgrounds de sección)
- Blanco: #FFFFFF (fondo principal de la mayoría de slides)
- Gris oscuro: #1A1A2E (texto principal, títulos)
- Gris medio: #4A4A68 (texto secundario, descripciones)
- Gris claro: #F5F5F7 (fondos alternos, cajas)
- Negro: #000000 (texto de alto contraste donde se necesite)

TIPOGRAFÍA:
- python-pptx no embebe Google Fonts fácilmente, así que usa fuentes seguras:
  - Títulos: "Calibri" bold, tamaño 28-36pt
  - Subtítulos: "Calibri" semibold, tamaño 18-24pt
  - Body: "Calibri", tamaño 14-16pt
  - Notas/captions: "Calibri Light", tamaño 11-12pt

REGLAS DE DISEÑO:
- Máximo 5-6 líneas de texto por diapositiva. Menos es más.
- Usar íconos Unicode cuando sea posible (✓, →, ●, ◉, ▸, ★, ⚡, 🔒, 📊).
- Degradados naranja en las barras superiores o laterales de cada slide.
- Bastante espacio en blanco (no saturar).
- Los datos en tablas o cajas con bordes redondeados (simular con shapes).
- Footer discreto en todas las slides: "Círculo de Crédito | Confidencial"
- Aspect ratio: 16:9 (widescreen).
- Cada slide debe tener una barra superior o lateral naranja como elemento 
  de identidad visual consistente.

TAMAÑO DE PRESENTACIÓN: Widescreen 16:9 (33.867cm x 19.05cm)

INSTRUCCIONES TÉCNICAS PARA python-pptx
================================================================================

LIBRERÍA: python-pptx
INSTALACIÓN: pip install python-pptx Pillow

ESTRUCTURA DEL CÓDIGO:
1. Crear presentación con aspect ratio 16:9
2. NO usar los layouts predefinidos de python-pptx (son feos).
   En su lugar, usar slides en blanco (layout 6 o blank) y construir
   cada elemento manualmente con shapes.
3. Para cada slide:
   a. Agregar un shape de barra superior naranja (rectángulo sin bordes)
   b. Agregar textboxes para título, subtítulo, body
   c. Agregar shapes (rectángulos redondeados) para cajas/cards
   d. Agregar tablas donde se indica
4. Para la slide de cronograma: insertar WorkSheet.png como imagen

HELPERS SUGERIDOS:
- Función para crear barra naranja superior consistente
- Función para crear footer "Círculo de Crédito | Confidencial"
- Función para crear una card/caja con título + texto
- Función para crear tabla con estilo consistente
- Función para crear número destacado (para KPIs)
- Constantes para colores (RGBColor)
